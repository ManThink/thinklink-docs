# 1. 组织管理

## 1.1. 组织权限
+ 每个用户账号**仅能作为管理员管理一个组织**。
+ 但可以以成员身份**加入多个其他组织**，并在权限范围内参与协作。

## 1.2. 新增组织
适用于希望创建新组织并担任其管理员的用户，注意用户新增组织后，会自动把admin账号添加进去，用于协助用户处理问题，如果不需要admin账号，请登录到该组织的系统页面后，在 系统管理->用户管理中 中手动删掉admin。

1. 进入 `组织管理` 页面，点击**新增组织** 按钮。
2. 填写以下必填信息（所有字段均需在系统内保持唯一）：

| 字段名称       | 说明 |
|------------| --- |
| **组织账号**   | 用于标识该组织的唯一编码，由小写字母、数字及下划线（_）组成。此账号将用于生成用户编码，并作为 MQTT 订阅中的 `tenant` 字段。 |
| **组织名称**   | 组织的显示名称，可用于对外展示。 |
| **MQTT密码** | 用于第三方平台通过 MQTT 协议订阅 ThinkLink 数据时的身份认证。请妥善保管该密码。 |


3. 确认信息无误后，点击提交，系统将创建新的组织，当前用户自动成为该组织的管理员。
