# 1. 角色管理
**菜单路径**：系统管理 → 角色管理

ThinkLink（TKL）默认提供 `admin` 角色，该角色拥有当前组织下所有功能的最高操作权限，不可删除。

用户可通过以下方式新增自定义角色：

1. 点击**“新增角色”** 按钮，弹出新增角色配置窗口。
2. 输入角色名称并描述（可选）。
3. 在权限配置区域通过勾选对应的菜单项，为该角色分配具体的访问权限。
4. 点击**“确定”** 完成角色创建。

创建完成后，该角色可在用户管理中被赋予指定用户，实现细粒度的权限控制。

> 【提示】建议根据实际业务需求合理划分角色权限，避免权限过度集中或不足。
