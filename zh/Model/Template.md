# 1. 模板
模板配置是，将多个模型配置为一个集合，用户在增加设备或者批量导入时，可以通过模板一键完成多种模型的选择和配置。

## 1.1. 属性
可以设置服务端属性和共享属性，设置完成后，可以在模型中调用对应的属性。设备通过模版添加配置时，会将设置好的服务端属性和共享属性添加到设备属性中。
